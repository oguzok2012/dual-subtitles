# Dual Subtitles Pro (DualSubs)

Браузерное расширение (Chrome / Manifest V3) для отображения двойных субтитров поверх любых HTML5 видеоплееров. 

Расширение спроектировано для корректной работы со сторонними "морскими" видеоплеерами (включая встраиваемые через iframe, такие как Kodik, Alloha и др.). Реализовано с учетом строгих лимитов производительности браузера и защиты от конфликтов с CSS-стилями родительских страниц.

## Инструкция по использованию
*   Откройте сайт с сериалом.
*   Выберите режим с субтитрами.
*   Выберите сначала русские, затем английские. Это необходимо для того, чтобы субтитры подтянулись в браузер. Затем можете их скрыть.
*   Откройте окно расширения. Настройте положение субтитров.
*   Нажмите кнопку "Применить к плееру".
*   Наслаждайтесь просмотром.
*   Действия необходимо повторять для каждой новой серии.

## Основной функционал

*   Одновременное отображение двух VTT/SRT-дорожек. 
*   Поддержка iframe-плееров с полной изоляцией верстки субтитров от стилей основного сайта.
*   Настройка задержки (в миллисекундах) для корректировки рассинхрона видео и текста.
*   Динамическое изменение размера шрифта и позиционирования слоев без перезагрузки страницы (Live Preview).
*   Автоматическое сохранение выбранных языков и настроек в `chrome.storage.local`.

## Установка

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/oguzok2012/dual-subtitles.git
   ```
2. Откройте браузер Chrome на странице управления расширениями: `chrome://extensions/`
3. Включите **Режим разработчика (Developer mode)**.
4. Нажмите **Загрузить распакованное расширение (Load unpacked)** и выберите директорию проекта.
5. Да, мне в падлу выкатывать расширение к магазин.

## Структура проекта

*   `manifest.json` — конфигурация расширения (Manifest V3).
*   `background.js` — Service Worker. Отвечает за сниффинг трафика (`chrome.webRequest`), обход CORS и координацию IPC-сообщений.
*   `content.js` — контроллер плеера. Управляет жизненным циклом событий видео, синхронизацией времени и кэшем индексов.
*   `renderer.js` — View-слой. Отвечает за инъекцию в DOM, позиционирование через rAF и защиту от DOM-Thrashing.
*   `parser.js` — десериализатор файлов `.vtt` и `.srt` с функционалом очистки сырых строк от HTML-тегов.
*   `ui.html` / `ui.js` — интерфейс пользователя (Popup) с поддержкой мгновенного применения настроек через внедрение микро-скриптов (`chrome.scripting.executeScript`).
